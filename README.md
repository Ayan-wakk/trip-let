## サービス概要
お出かけしたスポットの情報・感想・注意点を共有できるWebアプリです。
実際の体験に基づいたリアルな情報を集め、次に行く人の不安を解消します。
写真やコメントを通じて、感動を共有し旅行者同士がゆるくつながれる場所を提供する。

投稿内容は様々です。
観光やレストラン、カフェや御朱印記録など。
非公開で登録すると自分の記録(日記)としても使えます。

## このサービスへの思い・作りたい理由
旅行に行くとき、ネット上の情報は広告や記事のまとめが多く、
実際の旅行者の「本音の体験談」にはなかなかたどり着けないと感じてきました。
以前、旅行前に参考にしていた情報が実際と大きく違って困った経験があり、
「リアルな旅行体験を残せる場所があればいいのに」と思ったのが企画のきっかけです。
また、SNSで投稿するほどではないけれど「誰かの役に立つかもしれない情報」を
気軽に共有できる場を作りたいという思いがあります。

## ユーザー層について
✦ 旅行前の人（10〜40代）
　→ 旅行先のリアルな情報を知りたいニーズが高く、特に個人手配の旅行者は判断材料を求めているため。
✦ 旅行が好きで情報を共有したい人
　→ 自分の体験談を形にして残したい人・写真を投稿したい人が一定数いるため。
✦ SNSほど拡散しない場所で情報を残したい人
　→ 不特定多数には見せたくないが、役立つ情報は残したいというニーズがあるため。

## サービスの利用イメージ
ユーザーは旅行後に写真と感想を投稿し、行った国・場所・注意点をシェアできます。
旅行前のユーザーは国名や地域で検索し、実体験にもとづく投稿を参考にすることで、
安心して旅行計画を立てられます。
お互いの経験が蓄積されることで、“実際に行った人の声がまとまっている旅行ノート” のような価値を提供します。

## ユーザーの獲得について
・SNS（Instagram / X）での情報発信
　旅行好きユーザーが多いコミュニティからの流入が見込める。
・友人・知人に使ってもらい口コミを広げる
・投稿キャンペーン（バッジ付与等）
　投稿数を初期に増やす仕組みを用意する。
・検索流入（SEO）
　国名＋体験談の検索に引っかかる形を想定（将来的な施策）。

## サービスの差別化ポイント・推しポイント
✦ 既存サービスとの違い
　・SNS（Instagram・TikTok）
　　｜写真中心で情報が流れてしまう
　　｜旅行者同士で「注意点」や「実体験」を体系的に探しにくい
　　　→ 本サービスは「実体験情報」に特化し、検索しやすく整理されている。
　・ブログ・まとめサイト
　　｜個人のブログは更新が少ない
　　｜企業記事は広告や提携が混ざることが多い
　　　→ このサービスは「ユーザーの本音ベース」で広告色が弱い。
  ・類似アプリ（例：Travy）
    |写真と短いコメントが中心で、実体験の詳細が分かりにくい
    |国内中心で海外情報は少なめ
    |スポット単位で投稿がバラつき、1日の流れが見えない
   　 → 本サービスは以下の点で差別化しています：
　　　　●行った時期・実体験・注意点まで詳しく共有できる
　　　　●その日に訪れたスポット一覧で1日の動きを把握できる
　　　　●世界中の旅行に対応し、国名・地域名で検索可能
　　　　●旅行スタイルが近いユーザーとゆるく繋がれる

✦ 継続的に利用したくなる仕組み
　・マイページが“自分の旅行記アーカイブ”として蓄積される
　・旅行前：保存・検索・フォローで情報収集
　・旅行中：訪れた場所をメモとして下書き保存
　・旅行後：投稿して旅の記録として残せる
　・バッジ・ランキングなど成長を実感できる機能

✦ 推しポイント（独自性）
　・写真＋注意点＋感想がセットになった「体験カード」形式で見やすい
　・旅行スタイルが近いユーザーを見つけやすく、
　　　　“ゆるくつながる小さな旅行コミュニティ”を形成できる

## 機能候補
　～MVPリリース～
　・ユーザー登録 / ログイン
　・投稿作成（写真5枚まで・国・場所・感想・注意点）
　・投稿一覧表示
　・投稿詳細ページ
　・国・地域での検索機能
　・自分の投稿一覧・CRUD

　～本リリースまでに追加したい機能～
　・いいね / 保存機能
　・コメント機能
　・投稿にタグ付け（例：グルメ、交通など）
　・フォロー機能（旅行ジャンルが近い人を見つけやすい）
　・地図上で投稿を可視化
  ・バッジ付与機能
  ・人気ユーザーランキング機能

　～本リリース後追加したい機能～
　・お気に入りの投稿をもとに旅のしおり作成機能
